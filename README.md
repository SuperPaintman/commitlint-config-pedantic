--------------------------------------------------------------------------------

## Installation

```sh
npm install commitlint-config-pedantic --save-dev
```


--------------------------------------------------------------------------------

## Usage

Extend your `commitlint.config.js`:

```js
module.exports = {
  extends: [
    'commitlint-config-pedantic'
  ]
};
```
